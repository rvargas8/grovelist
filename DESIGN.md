# The Grove List — Design System

> Guiding principle: **typographic warmth over visual decoration**.
> Every element should feel authored by a person, not generated by a template.
> References: *Elements of Typographic Style* (Bringhurst), *The Design of Everyday Things* (Norman).

---

## Aesthetic Direction

Editorial, unhurried, local. Think a well-designed neighborhood newspaper, not a SaaS dashboard. The site should feel like it was made by someone who actually lives here — not a startup that parachuted in.

**Not this:** colored gradient hero blocks, emoji icons, 3-column feature grids, "clean modern UI"
**This:** tight typographic hierarchy, warm off-white backgrounds, restrained use of color, prose that reads like a recommendation from a friend

---

## Typography

| Role | Font | Size | Weight | Notes |
|------|------|------|--------|-------|
| Display / Heading | Lora (serif) | h1: 2.75rem, h2: 1.375rem, h3: 1.0625rem | 600 | Letter-spacing: -0.02em at large sizes (Bringhurst) |
| Body | Source Sans 3 (sans) | 1rem | 400 | Line height: 1.6 |
| Tagline / Eyebrow | Lora italic | 0.9375rem | 400 | Used for subheadings and mover banner |
| Muted / Caption | Source Sans 3 | 0.875rem | 400 | color-text-muted |

**Measure (line length):** 65ch — comfortable reading width per Bringhurst.
**Fallbacks:** Georgia (heading), system-ui (body).

## Color Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `--color-text` | `#2d2d2d` | Primary text — near-black, not pure black |
| `--color-text-muted` | `#6b6259` | Secondary text, captions, metadata |
| `--color-bg` | `#faf7f2` | Page background — warm off-white |
| `--color-bg-card` | `#ffffff` | Card and header background |
| `--color-border` | `#e6e0d8` | Borders — warm gray |
| `--color-accent` | `#bf5a38` | CTAs, links, interactive — terracotta |
| `--color-accent-hover` | `#a8492d` | Hover state for accent |
| `--color-featured` | `#c17f24` | Grove Pick gold — amber |
| `--color-featured-bg` | `#fdf6e8` | Featured card background — pale amber |

**Palette logic:** Warm earth tones. The accent (terracotta) and featured (amber) are both drawn from grove / citrus imagery. Never use cool blues or grays — they break the warmth.

## Border Radius

`--radius: 2px` — nearly square. Deliberate. Rounded corners feel generic; near-square corners feel editorial and considered.

## Components

### Header / Masthead
- Logo stacked above title (column, centered)
- Logo: 96px tall × auto width
- Title: Lora 2.75rem, `--color-featured`, letter-spacing -0.02em
- Tagline: Lora italic 0.9375rem, `--color-text-muted`
- No CTA in the header — the header is pure brand

### Listing Cards
- `--color-bg-card` background, 1px `--color-border` border, `--radius` corners
- Grove Pick (featured) cards: `--color-featured-bg` background, amber border glow animation
- **Grove Pick hover:** `translateY(-6px)` lift + deeper amber shadow — exclusive to Grove Picks
- Regular cards: subtle shadow on hover only

### About Section
- Text-only cards — no colored image blocks, no emoji
- Title + body + optional "Learn more →" link
- The prose is the design

### Empty State
- Centered, `--spacing-2xl` padding
- Dynamic message: `No matches for "[query]" — try a different search, or browse by category below.`
- "Clear search" button: ghost style, `--color-border` border

### Search
- Visible label: "Find a local business" (Lora, 1.25rem)
- Pill-style input + button
- Centered in `--color-bg-card` section

---

## Motion

- Transition: `0.25s ease` for hover states (box-shadow, border-color, transform)
- Grove Pick glow: `featured-glow` keyframe, 3s infinite — pauses on hover
- `prefers-reduced-motion: no-preference` gates the glow animation

---

## Accessibility

- Minimum tap target: 44px (`--tap-min`)
- All interactive elements: visible `:focus-visible` outline, `outline-offset: 2px`
- Screen-reader-only labels: `.sr-only` utility class
- Color contrast: all text meets WCAG AA against respective backgrounds

---

## Page Flow (index.html)

1. Header / Masthead
2. Search ("Find a local business")
3. Category filters
4. Grove Picks (featured listings)
5. Worth Discovering (non-featured listings)
6. "New to the Southshore area?" banner
7. About section (text-only cards)
8. "List Your Business" CTA
9. Footer

---

## What NOT to do

- No colored gradient image blocks ("hero cards with icons")
- No emoji used as decorative icons
- No pure black (`#000`) — use `--color-text`
- No cool-toned colors (blue, gray) — warm palette only
- No duplicate CTAs — one conversion point per action
- No `px` units in layout — use `rem` and CSS custom properties
